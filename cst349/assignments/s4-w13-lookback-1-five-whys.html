How this works · Read first

The Look-back workflow — seven steps.

  1. Fill out the form activity below. The cognitive work has to be yours. An AI sparring partner is available on some questions if you want a thinking partner — it asks questions back, it won't write your answer for you.
  2. At the end, click Download Responses. Copy the markdown field from the exported JSON (or use the activity's markdown summary).
  3. Create a Google Doc titled "[Your name] — Sprint 4 Look-backs". This is a new doc, separate from your Sprint 4 Journal.
  4. Paste your summary into the doc.
  5. Make any additions or edits you want directly in the doc. This is your space — expand, clarify, push harder on anything.
  6. Download the doc as a PDF. File → Download → PDF Document.
  7. Submit the PDF here on Canvas. PDF only.

Important: Your Look-back doc is separate from your Sprint 4 Journal. Each new look-back you do gets added to this same Look-back doc, and each PDF you submit contains everything you've written so far. Do not submit text. Do not submit a different file format.
