Path B: Create a Document Package

For those who want to see strategic document creation

The Task

Ask Claude to create a coordinated set of documents for different audiences or purposes. The goal is to experience delegating complex, multi-part document creation where context and consistency matter.

Ideas to Try

  • Communication package ‚Äî Same announcement tailored for 3 different audiences (students, faculty, industry partners)
  • Workshop series ‚Äî Design a 4-session workshop with facilitator guides, participant materials, and assessment rubrics
  • Strategic analysis ‚Äî Analyze a decision from multiple perspectives with recommendations
  • Onboarding kit ‚Äî Create documentation for bringing someone new into a complex project or role

Sample Prompts

  • "I need to communicate [this change] to three different groups who care about different things. Create tailored versions for each."
  • "Design a workshop series on [topic]. I need facilitator guides, participant handouts, and a way to know if it worked."
  • "Help me think through [this decision] from multiple angles. What would different stakeholders care about? What are the tradeoffs?"

üí° Tips for Success

  • Start with the end in mind. Describe what you want to exist, not the steps to get there.
  • Let Claude make choices. You can always redirect‚Äîthat's part of the experience.
  • Notice when you want to intervene. What prompts that instinct? Is it helpful?
  • Iterate through conversation. "That's close, but change X" is more powerful than detailed specifications upfront.
  • Don't worry about perfection. The goal is to experience the collaboration, not produce a polished deliverable.
  • Write down what surprises you. These observations will fuel the debrief.
