Monday, February 2

Beyond What You've Tried Before

Working at a Higher Level of Abstraction

Session Overview

Time 1:00pm - 5:00pm (4 hours) Format Team connection + Paired hands-on exercise with Claude Code Goal Connect as humans, then experience what it means to work at a strategic/delegation level with AI

Detailed Agenda designed

1:00 - 1:30pm

Connection Activity

Start with human connection before diving into strategy. 9 themed cards—pick one, answer a question, team guesses who wrote it.

→ Open Connection Cards

Interactive flip cards with questions for the team to answer and guess.

1:30 - 5:00pm

Hands-On: Human-AI Collaboration

Experience what it means to work at a strategic/delegation level with AI through paired exercises with Claude Code.

1:30 - 1:45pm

Setup

Brief orientation to Claude Code. Not a tutorial—just enough that no one is stuck on mechanics. Introduce the two paths for the exercise.

Reference Demonstrations

Examples of what's possible—apps built, documents created, explorations completed.

→ View Reference Demonstrations

Activity Guide

Detailed instructions for the hands-on exercise: choose your path, setup steps, discussion prompts.

→ Open Activity Guide

1:45 - 2:10pm

Phase 1: Beyond What You've Tried Before (Pairs)

Choose one of two paths:

Path A: Build and Deploy an App

Ask Claude Code to build and deploy a working web application. Describe what you want—something useful to you—and let Claude handle the implementation.

Path B: Create a Document Package

Ask Claude to create three funding packages for three different funding sources—each with a budget request and a contingency version if funding is halved.

While Claude works, discuss: What is different about giving a task like this to AI vs. how you have used it in the past?

2:10 - 2:20pm

Quick Capture

Before break, each person writes down (on paper) two things they noticed. No discussion yet—just capture.

2:20 - 2:40pm

Break

2:40 - 3:20pm

Phase 2: Directed Iteration (Pairs, Switch Drivers)

Switch who is driving. Continue with your chosen path—iterate and improve what you built.

Discuss and write down: What aspects of your cognitive capabilities are you tapping into? Is it possible to identify and articulate them to someone else?

3:20 - 3:30pm

Individual Reflection

Before debrief, each person writes down:

Reflection Questions

  1. What did the human provide that Claude couldn't?
  2. What capability would someone need to do this well?
  3. One moment that shifted how you think about this.

3:30 - 5:00pm

Debrief (Three Rounds)

Round 1 — What happened? (20 min)

Each person shares one specific moment: "Claude did X and I realized Y." Go around the room. No discussion yet—just collection.

Round 2 — What patterns? (20 min)

What did we hear across people? Where did Claude make choices we didn't anticipate? Where did human direction matter most?

Round 3 — So what? (50 min)

If this is how work is changing, what does it mean for what we teach? What capabilities did you need that you didn't have? What would a student need to do this well?
