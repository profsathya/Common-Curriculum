Monday, February 2

Experiencing the Shift

Working at a Higher Level of Abstraction

Session Overview

Time 1:00pm - 5:00pm (4 hours) Format Team connection + Paired hands-on exercise with Claude Goal Connect as humans, then experience what it means to work at a strategic/delegation level with AI

Detailed Agenda designed

1:00 - 2:00pm

Team Activity: Connection Cards

Start with human connection before diving into strategy. A fun activity to learn something new about each other.

→ Open Team Activity Page

Interactive cards with questions for the team to answer and guess.

2:00 - 5:00pm

Demonstration: Human-AI Collaboration

Experience what it means to work at a strategic/delegation level with AI through a hands-on paired exercise.

2:00 - 2:15pm

Setup

Brief orientation to the Claude desktop app. Not a tutorial—just enough that no one is stuck on mechanics.

Demo: Human-AI Collaboration in Action

In ~45 minutes of collaborative conversation with Claude Code, I built and deployed 5 fully-functional web apps to demonstrate AI-assisted development for my students:

  • Mind Grid — A pattern memory game with 30+ levels and 6 evolving mechanics
  • DevPath — A productivity tracker for CS students with habit tracking and skill roadmaps
  • SecureScan — A website security analyzer with letter grades and fix recommendations
  • Abstraction Explorer — An interactive educational journey where students experience why abstraction matters (with hands-on challenges at each level)
  • Build Story — A visual presentation of how the entire session unfolded

See here: ai-demonstrations.netlify.app →

2:15 - 2:40pm

Phase 1: The Ambitious Ask (Pairs)

All pairs do the same task:

Task

Ask Claude to create three funding packages for three different funding sources—each with a budget request and a contingency version if funding is halved.

While Claude works, discuss: What is different about giving a task like this to AI vs. how you have used it in the past?

2:40 - 2:50pm

Quick Capture

Before break, each person writes down (on paper) two things they noticed. No discussion yet—just capture.

2:50 - 3:10pm

Break

3:10 - 3:50pm

Phase 2: Directed Iteration (Pairs, Switch Drivers)

Switch who is driving. Pick one funding audience and iterate to make that package better.

Discuss and write down: What aspects of your cognitive capabilities are you tapping into? Is it possible to identify and articulate them to someone else?

3:50 - 4:00pm

Individual Reflection

Before debrief, each person writes down:

Reflection Questions

  1. What did the human provide that Claude couldn't?
  2. What capability would someone need to do this well?
  3. One moment that shifted how you think about this.

4:00 - 5:00pm

Debrief (Three Rounds)

Round 1 — What happened? (15 min)

Each person shares one specific moment: "Claude did X and I realized Y." Go around the room. No discussion yet—just collection.

Round 2 — What patterns? (15 min)

What did we hear across people? Where did Claude make choices we didn't anticipate? Where did human direction matter most?

Round 3 — So what? (30 min)

If this is how work is changing, what does it mean for what we teach? What capabilities did you need that you didn't have? What would a student need to do this well?
