Reflect: what do you actually need to learn here?

Slow down before you build. This sets how you’ll use AI for the rest of the program.
Learning outcome
You can tell what you genuinely need to learn from what you can safely skip — and you reach for AI to learn, not to answer for you.

The activity

Before fixing anything, map your own learning by hand. The point is to ask “what do I actually need to learn here?” before you start prompting.

How to reflect

  1. By hand, list what you’d need to understand to make a useful contribution here.
  2. Mark what’s essential now versus what you can pick up later.
  3. Only then use AI to go deep on the essentials — as learning infrastructure, not an answer machine.
